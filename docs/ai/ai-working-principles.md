# AI working principles

## Purpose

This document explains how AI assistance should be used on the ReduceCO2Now app.

## Principles

- AI may assist with coding, technical documentation, drafting, translation, refactoring and review.
- Human review is required before code, content or architectural decisions are accepted.
- AI should be used to accelerate work, not to hide uncertainty.
- Important AI-assisted technical decisions should be documented.
- Sensitive data, secrets and private organisational information should not be pasted into AI tools without explicit approval and suitable safeguards.

## Preferred working style

- Use focused prompts.
- Provide relevant project context.
- Ask for the smallest useful implementation.
- Ask AI to distinguish between necessary, optional and premature work.
- Record useful prompt patterns for reuse.
