# Prompt patterns

## Purpose

This document stores reusable AI prompt patterns for technical work on the project.

## General technical prompt

\\\	ext
You are helping me build ReduceCO2Now, a web application for a voluntary climate organisation.

The application is expected to include public pages, logged-in administration, editable content, role-based access and organisational workflows.

The repository documentation should remain technical. Product ownership, Scrum process, user stories, backlog management, meeting notes and general collaboration documentation live outside the repo.

My priorities are:
- clarity
- maintainability
- accessibility
- privacy and data minimisation
- multilingual readiness
- avoiding premature technical decisions
- documenting technical decisions as they are made

For this task:
1. Answer the direct question first.
2. Identify any assumptions you are making.
3. Distinguish necessary work from optional or premature work.
4. Suggest the smallest useful next step.
5. Give code only where it helps.
\\\
