# Roles and permissions

## Purpose

This document records the technical permission model for the ReduceCO2Now application.

## Initial considerations

The application is expected to have logged-in users with different levels of access.

Possible role areas include:

- site administration
- content editing
- publishing or approval
- event management
- campaign or initiative management
- user management
