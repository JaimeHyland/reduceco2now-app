# The ReduceCO2Now Website

This repository contains the code and technical documentation for the ReduceCO2Now web application.

ReduceCO2Now is a voluntary organisation. The application is expected to include a public website, logged-in administration, editable content, role-based access, and organisational workflows.

## Current status

Initial technical planning and scaffold.
